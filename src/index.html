Gulp BrowserSync

BrowserSync가 제공하는 가벼운 개발 서버는 웹 클라이언트 작업시 결과를 쉽게 빠르게 확인할 수 있게 도와줍니다.

코드의 변경사항을 여러 장비에 빠르게 반영하며, 관리자 화면에서 보다 많은 기능을 제공합니다.

BrowserSync는 Node.js 기반의 어플리케이션이라서 Gulp와 매끄럽게 연동해서 사용할 수 있습니다.

주요 기능

  1. Mini Web Server
  2. Sync (여러 장비에서 동일한 화면이 보이도록 Sync하는 기능)
  3. LiveReload
  4. Admin (http://localhost:3001)
    • Sync Options
    • History
    • Plugins
    • Remote Debug
    • Network Throttle
    • Help

개발중인 다른 서버에 브라우저싱크 연동

다른 개발 서버에 프록시 연결 하여 브라우저 싱크 사용할 수 있다.
준비된 예제 소스의 "gulpfile.js" 보면 쉽게 알 수 있다.

실행 (Command Line Interface)

  1. npm install
  2. gulp
  3. 자동으로 브라우저에 설정한 index.html 화면이 출력된다.
